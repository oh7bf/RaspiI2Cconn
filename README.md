RaspiI2Cconn
============

I2C connection board with modular 4P4C connectors for Raspberry Pi.

Parts
=====

```textile
CONN1-5         4P4C modular
F1              polyfuse 100 mA (or smaller)
J1,J3,J4,J5,J6  2 pins header
J2              2x3 pins header
R1,R2           100
Z1,Z2           zener 3V3
```
